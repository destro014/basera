# User Flows

## Renter Journey

Registration  
→ Profile creation  
→ Browse listings  
→ Filter listings  
→ View listing details  
→ Save listing  
→ Send interest request  
→ Owner approval  
→ Chat  
→ Visit scheduling  
→ Assignment  
→ Agreement signing  
→ Move-in  
→ Monthly billing  
→ Payment  
→ Review owner  
→ Move-out
