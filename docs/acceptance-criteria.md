# Acceptance Criteria

## Authentication

User can:

- enter phone number
- receive OTP
- verify OTP
- resend OTP

Invalid OTP shows error.

User proceeds to role selection after verification.

---

## Profile Creation

Renter profile requires:

- full name
- phone
- email
- occupation
- family size
- pets
- smoking status

Owner profile requires:

- full name
- phone
- email
- address
- ID verification
- bank/payment details

Profile cannot be completed if required fields missing.

---

## Listing Creation

Owner can:

- create listing
- upload photos
- upload videos
- set rent
- set utilities
- add rules

Listing cannot publish without required fields.

---

## Listing Discovery

Renter can:

- browse listings
- view list view
- view map view
- apply filters
- save listings

Exact address must remain hidden.

---

## Interest

Renter can send interest request.

Owner receives notification.

Owner can:

- accept
- reject

Chat unlocks only after approval.

---

## Agreement

Agreement preview available.

Both parties must sign.

OTP required.

Agreement locked after signing.

Agreement PDF generated.

---

## Billing

Owner can generate invoice.

Invoice includes rent and utilities.

Renter receives notification.

Invoice visible in renter dashboard.

---

## Payments

Renter can pay using:

- eSewa
- Fonepay

Owner can mark cash payment.

Partial payments allowed.

Payment receipts generated.

---

## Move-Out

Renter can request move-out.

Owner approves request.

Checklist completed.

Deposit refund recorded.

Tenancy archived.
